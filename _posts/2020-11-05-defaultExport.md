The "**defaultExport**"-section determines all export-settings regarding the **snapADDY Grabber**. 
An empty defaultExport looks like this:

{% highlight javascript %}{% raw %}
"defaultExport": {
    "mappings": {
        "*": {}
    },
    "workflows": {
        "*": {}
    },
    "components": {
        "*": {}
    }
}
{% endraw %}{% endhighlight %}
