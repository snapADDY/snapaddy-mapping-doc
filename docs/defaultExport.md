# Default Export

The "**`defaultExport`**"-section determines all export-settings regarding **DataQuality** and **BusinessCards**.
An empty `defaultExport` looks like this:

```json
{
  "defaultExport": {
    "mappings": {
      "*": {}
    },
    "workflows": {
      "*": {}
    },
    "components": {
      "*": {}
    }
  }
}
```
