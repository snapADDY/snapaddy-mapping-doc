# Introduction

snapAddy allows you to use pre-built, out-of-the-box interfaces with your CRM system for a plug-and-play experience. Simply connect to your CRM system using one of the snapAddy products and instantly create or update contact information for default fields like name, company, and address.

Many customers utilize custom fields or custom entities. If you need to export information beyond the snapAddy standard, you can do so with an individual mapping. This enables you to connect to custom fields and entities, implement custom functions for the snapAddy apps, or display information differently within the snapAddy apps (e.g., show additional fields).
